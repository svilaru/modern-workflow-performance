Parcel

Parcel es un popular empaquetador de módulos JavaScript popular por su fácil instalación, facilidad de uso y su "configuración cero". Este module bundler permite desarrollar partes del sitio o aplicación web como módulos, permitiendo la reutilización del código.

Parcel toma los archivos JS (o módulos) separados y reutilizables y los "agrupa" en un sólo archivo para servir al navegador, así como para minimizar la salida. Esta característica mejora directamente el rendimiento del sitio web, ya que el navegador no necesita cargar multitud de archivos individualmente. Además, Parcel también se encarga de otras tareas que incluyen:

  • Montar un servidor local
  • Compilar y minificar HTML, CSS y assets
  • Transpilar Javascript
  • Recargar módulos en caliente
  • Code Splitting

Instalación

Partiendo de la premisa que se debe tener instalado Node.js y Node Package Manager (npm) en nuestro SO, y por otra parte, que se debe tener un directorio básico estructurado de nuestro proyecto con el archivo package.json en la raíz del mismo, se procede a introducir el comando npm install -g parcel-bundler en el terminal.

Una vez completada la instalación, ya se puede montar un servidor local y comprobar cómo sin ninguna configuración extra, Parcel compila, minifica, transpila el código, etc.

Conclusiones

A día de hoy, Parcel se ha convertido en un module-bundler popular para un desarrollo moderno de aplicaciones o sitios web. Con este empaquetador el desarrollador evita la instalación de plugins y configuraciones extra, a no ser que sea que no se pueda evitar, maximizando el tiempo de desarrollo y producción.
