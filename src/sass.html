SASS

Sass es el preprocesador CSS más usado y demandado debido al gran apoyo que recibe de la comunidad. Entre sus principales características tenemos:

  • Uso de variables
  • Permite anidar declaraciones
  • Permite archivos parciales de Sass
  • Uso de mixins
  • Permite extender y/o la herencia de reglas
  • Permite usar operadores aritméticos
CSS on its own can be fun, but stylesheets are getting larger, more complex, and harder to maintain. This is where a preprocessor can help. Sass lets you use features that don't exist in CSS yet like variables, nesting, mixins, inheritance and other nifty goodies that make writing CSS fun again.

Instalación

Hay varios modos de instalar SASS, mediante la línea de comandos o mediante alguna aplicación. En este caso realizaremos los pasos mediante la línea de comandos para instalar el preprocesador como una dependencia de un proyecto basado en Node.js

Mediante el comando npm install -g sass se instalará SASS globalmente en el SO y ya se podrá hacer uso de este potente preprocesador.

*Para instalar SASS mediante una aplicación se puede acceder a la documentación oficial https://sass-lang.com/install
