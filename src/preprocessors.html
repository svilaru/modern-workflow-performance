Preprocesadores

Los preprocesadores CSS son herramientas para los desarrolladores de sitios web, que permiten traducir un código de hojas de estilo no estándar, específico del preprocesador en cuestión, a un código CSS estándar, entendible por los navegadores.

Con los preprocesadores básicamente se extienden las funcionalidades de CSS gracias a la posibilidad de utilizar variables, anidación de selectores, funciones (denominadas mixins), etc. Al desarrollar con un preprocesador se consigue flexibilidad en el momento del desarrollo, reutilizar código, y en definitiva obtener un código más sencillo de mantener, gracias a una mejor organización del código y la posibilidad de editar variables para que afecten a decenas, o cientos, de lugares del código CSS generado.

Ventajas de usar preprocesadores

  • Reutilización de código.
  • Reducción del tiempo de desarrollo.
  • Código más organizado.
  • Proyectos más fáciles de mantener.
  • Características dinámicas: variables, clases, funciones, etc.
